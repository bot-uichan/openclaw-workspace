# discord-mutual-mention

## 目的
- Discord上で、指定された複数ユーザーへ確実に通知を飛ばす。
- 名前文字列ではなく、`<@USER_ID>` 形式で明示メンションする。

## 実行手順
1. メンション対象を確定する。
   - 可能なら最初から `Discord ID` をもらう。
   - 不明なら、ユーザーに ID 提示を依頼する。
2. 本文を短く作る。
   - 要件（例: PRリンク、レビュー依頼）を1〜3行で書く。
3. `message` ツールで送信する。
   - `action: send`
   - `channel: discord`
   - `target: <thread/channel id>`
   - `message`: `<@id1> <@id2> ...` を含む本文
4. 送信後、必要なら一言で完了報告する。

## 送信テンプレート
```text
PR: <URL>
<@1473325761011257364> <@1473669278006775828> レビューお願いします！
```

## ルール
- メンションは必ずID形式で行う（表示名のみは不可）。
- 過剰通知を避ける（同内容の連投をしない）。
- 追加通知が必要な場合は「修正文で再通知」を1回だけ行う。
